# XGram Desktop

![Логотип XGram](.github/XGram.png)

XGram — настольный клиент Telegram, ориентированный на работу с несколькими аккаунтами, управление приватностью и настройку интерфейса.

> XGram находится в активной разработке. Перед проверкой новых сборок создавайте резервную копию локальных данных.

## Основные возможности

- XGram Preferences с отдельными переключателями функций
- хаб для работы с несколькими аккаунтами
- общие закрепы между аккаунтами
- глобальный и отдельный для каждого чата режим призрака
- история удалённых сообщений и исчезающих медиа
- отключаемый виджет курса криптовалют
- массовые действия с контактами и аккаунтами
- расширенная информация профиля и настройка интерфейса

## Загрузка

Стабильные и тестовые сборки публикуются в разделе **Releases** этого репозитория.

Не загружайте XGram из непроверенных источников. XGram никогда не попросит отправить или опубликовать вашу папку `tdata`.
