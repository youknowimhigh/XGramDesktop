# XGram Desktop

![XGram logo](.github/XGram.png)

XGram is a desktop Telegram client focused on multi-account workflows, privacy controls and interface customization.

> XGram is under active development. Back up your local data before testing new builds.

## Highlights

- XGram Preferences with per-feature controls
- Account Hub for working with multiple accounts
- Shared pins across accounts
- Global and per-chat Ghost Mode
- Deleted-message and expiring-media history
- Optional cryptocurrency price widget
- Contact and account batch actions
- Extended profile information and interface customization

## Downloads

Stable and test builds are published on the repository's **Releases** page.

Never download XGram builds from untrusted mirrors. XGram will never ask you to upload or share your `tdata` directory.
